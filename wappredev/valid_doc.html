二、完整方法

input标签 属性

检查表达式 [ valid-rule ] 必须带有该属性

example:valid-rule='en5-10'英文5~10位

en英文

zh中文

nb整数

!z非零整数

**任意

pw密码

ph电话

ma邮箱

id身份证号

fl浮点数，小数点后两位

an正整数和正小数

pn整数和小数

!s中文、数字、英文。（特殊字符除外 .~!@#$%^+*&/?|:{}()';="）

!t中文、数字、英文。（部分特殊字符除外 $'"）

zp大于等于0的整数和小数（小数默认两位）

ze大于0的正整数

zi大于等于0的正整数

ap大于0的正整数和浮点数 或不填

输入为空时的弹窗提示 [ valid-emptyAlert ]

调用checkValid时，input的值为空时，弹窗提示该属性的值，该值为空时没有弹窗

不符合检验的弹窗提示 [ valid-errorAlert ]

调用checkValid时，input的值不符合检验时，弹窗提示该属性的值，该值为空时没有弹窗

三、自定义绑定

用于自定义事件的序号 [ valid-index ]

该序号与pushfunctions传入值的index对应，用于绑定事件。

input valid-index='5'


pushfunctions([array])传入数组
array = [{index:index,ValidEmpty:function(){}},{index:index,ValidEmpty:function(){}},{index:index,ValidEmpty:function(){}}]
te.pushfunctions
([
{
index:'1',
ValidEmpty : function(){},
ValidClick : function(){},
ValidBlur : function(){},
ValidError : function(){}
},
{
index:'5', //对应valid-index的值
ValidEmpty : function(){},//空值时调用(可选)
ValidClick : function(){},//点击时调用(可选)
ValidBlur : function(){},//失去焦点时调用(可选)
ValidError : function(){}//检验错误时调用(可选)
}
]);
example：
引用模块
var VA = require('../modules/yue_ui/valid/index');
传入父元素节点
var te = new VA($('#node_here'));
te.checkValid();
