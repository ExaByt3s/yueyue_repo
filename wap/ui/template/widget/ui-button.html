/********大小*******/

ui-button-block 为block元素，100%

ui-button-size-large 为高度 45px 按钮

ui-button-size-middle 为高度 35px 按钮

ui-button-size-min 为高度 30px 按钮

/********圆角*******/

ui-button-size-radius-large 圆角为90px

/********配色*******/

ui-button-primary 默认色粉色

ui-button-color2 绿色

ui-button-color3 黄色

ui-button-nocolor 没有背景色

ui-button-dark 不可以点击黑色
